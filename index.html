Como usar

  1. Abra o Discord no navegador e vá no canal/DM.
  2. Clique no ícone (lixeira) e abra o painel do Diskill.
  3. Preencha Autor / Servidor / Canal (ou use os botões “me/current”).
  4. Clique em Apagar e acompanhe o log.

Notas importantes

  • Use com cuidado: exclusões são permanentes.
  • O script tenta apagar somente mensagens do seu usuário.
  • Se bater rate limit, aumente os delays.
Por que às vezes dá erro/404 em algumas mensagens?

Isso pode acontecer quando o Discord não permite apagar certas mensagens (ex.: não são suas, threads arquivadas, permissões, etc.). O script tenta contornar pulando o que não dá.

O Discord muda a interface com frequência?

Sim. Mudanças no site/API podem exigir atualização do script. Se quebrar, abra uma Issue no GitHub.
